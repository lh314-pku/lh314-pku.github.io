# 电脑小队AI培训主页

该页面是课程所使用的资料和网页链接整合。

如果关于培训内容有想法/建议，或者有什么问题想要解答，请发邮件 to 2400013085@stu.pku.edu.cn or 在群里加我微信。

AI技术这门课在国内开的很少，网上的教学也是零散的，或者说各有侧重（至少我只在研究生院那边听到过有类似的课，叫AI FOR SCIENCE，有兴趣可以关注一下北京大学研究生会的公众号）。

~~（偌大的学校竟无一课来教！）~~

所以这个培训的目的在于，让大家能够接触到实际工作中用得到的AI技术，例如接入大模型API、配置MCP Server、搭建知识库和Agent、多智能体协作、提示词工程……当下许多主流的AI技术公司或者互联网公司内部都在以AI为下一代产品（例如前几天在商场看到的AI牙刷，何意味……），例如Siri、小艺、飞书、Notion甚至抖音内内置的豆包（你北也有自己的大模型，如小北）。

AI势必是下一个时代不可或缺的技术手段，所以我们更需要知道如何用AI，不至于学完AI引论、CV、机器学习等等专业课，知道了大模型的各种原理和机制（什么Transformer、RAG、Attention……），结果参与工作后写代码给个BUG还是把代码CTRL+CV进去和AI扯皮。

总而言之，AI可以说是一门必修课，不论是原理还是使用方法。我们的培训内容主要有：

## [Chapter1](https://lh314-pku.github.io/training/Chapter1)

AI基础与工具概览。

## Chapter2

接口、API介绍以及调用

## Chapter3

MCP Server 和 知识库搭建

## Chapter4

Prompt 与 提示词工程

## Chapter5

Agent 简介与搭建

## Chapter6

多智能体协作
